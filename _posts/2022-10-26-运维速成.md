## linux命令

### 网络

```shell
ping # 判断是否能连接指定ip

telnet # 远程登录主机

netstat # 打印整个系统的网络情况

ifconfig # 配置和显示Linux内核中网络接口的网络参数

route # 命令用来显示并设置Linux内核中的网络路由表

host # 检测域名系统工作是否正常
```

### 进程

```shell
# 查看内存
free -h
free -m
# 查看进程占用内存

ps -ef | grep # 先找进程号
top  # 查看进程动态信息

ps aux # 所有进程信息

ps a #当前终端的所有进程

jobs # 当前终端所有后台进程

kill #s 杀死进程
```
